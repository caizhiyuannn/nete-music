# nete-music
基于Vue2, vue-router, vuex全家桶开发的网页版播放器。
网易云音乐后端API，本项目仅供学习使用。

感谢 [NeteaseCloudMusicApi](https://binaryify.github.io/NeteaseCloudMusicApi) 后端大佬API贡献！

## 使用说明

本地运行需要后端支持。

```bash
# 后端
cd NeteaseCloudMusicApi
npm install
npm start

# 前端
cd frontend
npm install
npm start

```

本地URL: http://localhost:8080/#/discovery
